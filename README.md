# Rubocop for CI

Run rubocop on your CI.

## Usage

Add this to the development/test group in your Gemfile:

```ruby
gem 'rubocop-ci', github: 'ad2games/rubocop-ci'
```

Run `bundle exec rake rubocop` before/after your tests on your CI.

## Changes
